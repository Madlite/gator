# Gator

Simple CLI RSS aggregator written in Go.

## Quick start

Prerequisites:
- Go (see `go.mod`)
- PostgreSQL

Create a config file at your home directory named `.gatorconfig.json` with this shape:

```json
{
  "db_url": "postgresql://user:pass@localhost:5432/gator?sslmode=disable",
  "current_user_name": ""
}
```

Build:
```sh
go build -o gator .
```

Run (examples):
- Register a user: `./gator register alice`
- Login: `./gator login alice`
- Add feed: `./gator addfeed <name> <rss_url>`
- Follow / unfollow / browse / list feeds: see handlers in `main.go` (`middlewareLoggedIn`, `handlerFollow`, `handlerUnfollow`, `handlerBrowse`, `handlerGetFeeds`)

Database code is generated by sqlc into `internal/database`. See the schema and queries:
- SQL config: `sqlc.yaml`
- Schema: `sql/schema`

If you need to run queries/migrations manually, use psql or your preferred migration tool against the DB URL in `.gatorconfig.json`.
